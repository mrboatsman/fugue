Module subsonic_bridge 

Source
Expand description

Bridges Subsonic API requests from Iroh QUIC streams to the axum router.

Each QUIC bi-directional stream carries one Subsonic API request/response. The client sends a JSON request (endpoint name + query params), this module constructs a synthetic HTTP request, calls the axum router as a tower Service, and streams the response back over the QUIC stream.

Special case: admin/events opens a long-lived event stream that pushes activity updates (now playing, friend online/offline) in real time.
